Module Imandra_syntax__Rewrite_parsetree

Rewriting Parsetree

This is used to rewrite the parsetree produced by Imandra's custom parser, into a parsetree that OCaml will accept. It also encodes some syntactic sugar (such as pattern matching against bigint constants) into normal OCaml constructs.

Intended for internal use.
