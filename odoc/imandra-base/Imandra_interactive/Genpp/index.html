Module Imandra_interactive.Genpp

Code Generation for Pretty Printers

This module provides pretty printers for all types defined in logic mode. The user should only concern themselves with eval, to_string, and to_file.

val to_string : unit -> string

Obtain the code for defining an OCaml module named "Pp" with pretty printers for all currently known types.

val to_file : ?⁠filename:string -> unit -> unit

Extract generator code into a valid OCaml file.

parameter filename

the file name (default "pp.ml")

val eval : ?⁠force:bool -> ?⁠quiet:bool -> unit -> unit

eval () is the same as Reflect.eval (to_string ()). After calling eval(), a module Pp should be accessible in the current Imandra environment.

parameter quiet

if true, evaluation is silent (default false)

parameter force

if true, force re-evaluation even if nothing changed (default false)

module Extend : sig ... end

Builtin Printers

Printers for builtin types. Do not call, they are intended for the generated code.
