Builtin Types and Functions

Imandra has a certain number of builtin types and functions it knows about. These builtins might have special properties w.r.t induction, or might translate into Z3 sorts and expressions in specific ways.

This module is used for listing builtin symbols, translating between (resolved) terms and Z3 expressions. A builtin function corresponds to some specific Z3 construct (e.g. an arithmetic operator in Smt.Arithmetic, or some other theory's defined symbols)

This should not be directly accessed by users, except for the recognizer functions such as Ty.is_int or Fun.is_and.
