Module Imandra_codegen__Gen_json

Code Generation for Json (de)serializers

This module provides serialization/deserialization for all types defined in logic mode.

A json module should have the following type:

module type JSON = sig
  type t =
    | Assoc of (string * t) list
    | Bool of bool
    | Int of int
    | List of t list
    | String of string
    | Null
end
val mk_plugin : mod_name:string -> unit -> Imandra_surface.Plugin.t
